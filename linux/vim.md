### vim 批量字符替换
```shell
:%s/from/to/g #批量替换所有字符 from：原字符串。to：新字符串
```

### vim 显示行号
```shell
vim /etc/vimrc
#新增（末行）
set nu
```

### vim 翻页
```shell
#整页翻页 
ctrl-f ctrl-b
#f就是forword b就是backward

#翻半页
ctrl-d ctlr-u
# d=down u=up

#滚一行
ctrl-e ctrl-y

zz #让光标所在的行居屏幕中央
zt #让光标所在的行居屏幕最上一行 t=top
zb #让光标所在的行居屏幕最下一行 b=bottom
```
